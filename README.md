# Devfolio

A full-stack developer portfolio platform where engineers can build, manage, and share their professional profiles. Each user gets a public profile page at `/u/[username]` and can export their portfolio as PDF or DOCX.

**Live:** [devfolio.uz](https://devfolio.uz) · **API Docs:** [api.devfolio.uz/docs](https://api.devfolio.uz/docs)

---

## Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js 16, React 19, TypeScript, Tailwind CSS v4 |
| Backend | NestJS 11, TypeScript, Passport.js |
| Database | PostgreSQL 16, Prisma ORM |
| Auth | JWT, Google OAuth2, GitHub OAuth |
| AI | OpenAI GPT-4o mini (bio generation) |
| Storage | AWS S3 (file uploads) |
| Infra | Docker, Nginx, VPS, GitHub Actions CI/CD |

## Getting Started

### Prerequisites

- Node.js 20+
- Docker & Docker Compose
- PostgreSQL (or use the Docker Compose setup)

### 1. Clone and configure

```bash
git clone https://github.com/ESardorbe/devfolio.git
cd devfolio
```

Copy the example env file and fill in the values:

```bash
cp .env.example .env
```

Required environment variables:

```env
# Database
POSTGRES_PASSWORD=your_secure_password

# JWT
JWT_SECRET=your_jwt_secret
JWT_REFRESH_SECRET=your_refresh_secret

# Google OAuth
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

# GitHub OAuth
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=

# Email (SMTP)
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USER=your@gmail.com
MAIL_PASS=your_app_password

# AWS S3
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_BUCKET_NAME=
AWS_REGION=

# OpenAI (for AI bio generation)
OPENAI_API_KEY=
```

### 2. Run with Docker

```bash
docker-compose up -d --build
```

- Frontend: http://localhost:3000
- API: http://localhost:4000/api
- Swagger: http://localhost:4000/docs

### 3. Run locally (development)

**API:**

```bash
cd apps/api
npm install
npx prisma migrate dev
npm run start:dev
```

**Web:**

```bash
cd apps/web
npm install
npm run dev
```

## Features

- **Authentication** — Email/password with OTP email verification, Google OAuth, GitHub OAuth, password reset flow
- **Portfolio sections** — Projects, Skills (with proficiency levels), Work Experience, Education, Certificates
- **AI bio generation** — One-click bio writing powered by OpenAI GPT-4o mini based on user's portfolio data
- **Public profiles** — Every user gets a shareable page at `/u/[username]`
- **Export** — Download your portfolio as PDF or DOCX
- **Profile analytics** — Track profile views by IP, country, and source
- **File uploads** — Avatars and certificate files stored on AWS S3
- **Dark / light theme** — Persisted per user via localStorage

## API Overview

Base URL: `https://api.devfolio.uz/api`

| Resource | Endpoints |
|---|---|
| Auth | `POST /auth/register`, `/auth/login`, `/auth/verify-email`, `/auth/forgot-password`, `/auth/reset-password` |
| OAuth | `GET /auth/google`, `/auth/github` |
| Users | `GET /users/:username`, `PATCH /users/me` |
| Projects | `GET/POST /projects`, `PATCH/DELETE /projects/:id` |
| Skills | `GET/POST /skills`, `PATCH/DELETE /skills/:id` |
| Experience | `GET/POST /experiences`, `PATCH/DELETE /experiences/:id` |
| Education | `GET/POST /educations`, `PATCH/DELETE /educations/:id` |
| Certificates | `GET/POST /certificates`, `PATCH/DELETE /certificates/:id` |
| Upload | `POST /upload/avatar`, `POST /upload/certificate` |
| AI | `POST /api/generate-bio` |

Full interactive docs available at `/docs` (Swagger UI, dev only).

## Deployment

The project deploys automatically to a VPS on every push to `main` via GitHub Actions.

### First-time VPS setup

```bash
ssh root@your-vps-ip 'bash -s' < scripts/vps-setup.sh
```

This installs Docker, Nginx, and Certbot, and generates a deploy SSH key.

### GitHub Secrets required

| Secret | Description |
|---|---|
| `VPS_HOST` | VPS IP address |
| `VPS_USER` | SSH user (e.g. `root`) |
| `VPS_SSH_KEY` | Private SSH key generated by setup script |

### VPS environment

The `.env` file is not tracked in git. On the VPS, create `/opt/devfolio/.env` with all required variables including `OPENAI_API_KEY`. The `docker-compose.yml` passes it to the web container at runtime.

### SSL

```bash
certbot --nginx -d devfolio.uz -d www.devfolio.uz -d api.devfolio.uz
```

### Manual deploy

```bash
cd /opt/devfolio
git pull origin main
docker-compose up -d --build --remove-orphans
docker image prune -f
```
